# linux-maintenance
Maintenance terminal commands for Linux

## Part 1
Use mainly these commands:

```
sudo apt clean

sudo apt update

sudo apt list --upgradable

sudo apt upgrade

sudo apt dist-upgrade

sudo apt autoremove --purge
```

## Part 2
```
sudo apt autoclean

sudo apt autoremove

sudo apt install baobab

sudo baobab

neofetch
```
