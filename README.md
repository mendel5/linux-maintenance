# linux-maintenance
Maintenance terminal commands for Linux

## Part 1
Use mainly these commands:

```
sudo apt clean

sudo apt-get autoremove --purge

sudo apt update

sudo apt upgrade

sudo apt dist-upgrade
```

## Part 2
```
sudo apt autoclean

sudo apt autoremove

sudo apt install baobab

sudo baobab

neofetch
```

## Other
```
#!/bin/bash
[ -d /sys/firmware/efi ] && echo UEFI || echo BIOS
```
